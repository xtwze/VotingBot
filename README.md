# 🗳 Telegram Voting Bot

## Структура проекта

```
telegram_bot/
├── main.py           # Точка входа, запуск бота и фоновый таск обновления
├── config.py         # Чтение .env (BOT_TOKEN, ADMIN_IDS)
├── .env              # Переменные окружения (не коммитить!)
├── states.py         # FSM-состояния (CreatePoll, VoteConfirm, Broadcast)
├── controller.py     # Построители клавиатур (InlineKeyboard)
├── db.py             # Все операции с SQLite через aiosqlite
├── text.py           # Все тексты сообщений
├── requirements.txt
└── handlers/
    ├── __init__.py
    ├── user.py       # Хэндлеры для обычных пользователей
    └── admin.py      # Хэндлеры для администраторов
```

## Быстрый старт

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Настроить .env
```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=123456789,987654321
```
> `ADMIN_IDS` — через запятую, узнать свой ID можно у @userinfobot

### 3. Запустить
```bash
python main.py
```

## Функции

### Обычный пользователь
- `/start` — приветствие + текущий опрос с кнопками голосования
- Голоса обновляются в кнопках в реальном времени (раз в 2 сек)
- Перед засчётом голоса — экран подтверждения
- Один голос на опрос

### Администратор (`/admin`)
| Кнопка | Действие |
|--------|----------|
| ➕ Создать голосование | Вводим тему → добавляем артистов → создаём |
| 👥 Список голосующих | Пагинированный список + `/delete_voice` |
| 📢 Рассылка | Пишем текст → предпросмотр → отправка всем |

- При новом голосе — уведомление с кнопкой 🗑 **Удалить голос** (удаляет из БД + блокирует юзера)
- При создании нового опроса — автоматически подводятся итоги предыдущего (топ-3)
- `/delete_voice @username` или `/delete_voice user_id` — аннулировать голос

### Команды
| Команда | Кто |
|---------|-----|
| `/start` | Все |
| `/admin` | Только ADMIN_IDS |
| `/delete_voice` | Только ADMIN_IDS |
